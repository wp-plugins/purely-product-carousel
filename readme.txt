Show a carousel that displays your latest products.

== Description ==

Various settings for number of columns and products to display.

== Installation ==

1. Upload purely-product-carousel.zip via the plugin upload menu, or use the built-in admin panel to locate "Purely Product Carousel"
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to widget areas and choose either an existing sidebar, or use the newly created Product Sidebar, that comes with this plugin and places a widget in the WooCommerce Product Page.
